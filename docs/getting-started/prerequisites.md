# 前提条件

AIエージェントを活用した研究を始める前に、必要な環境と基本的な知識を確認しましょう。

## 🖥️ システム要件

#### レベル1: 既存エージェントの利用
- **OS**: Windows、macOS、Linux すべて対応
- **メモリ**: 特別な要件なし（一般的なWebブラウザが動作する環境）
- **ストレージ**: 特別な要件なし

#### レベル2以降: Docker環境が必要
- **OS**: macOS、Linux を推奨（Dockerの安定性のため）
    - WindowsでもDockerが正常に動作すれば利用可能ですが、トラブルが発生する可能性があります
- **メモリ**: 8GB以上（推奨: 16GB以上）
- **ストレージ**: 10GB以上の空き容量

## 🛠️ 必須ソフトウェア

### レベル1: 既存エージェントの利用
- **Webブラウザ**: 最新版推奨

### レベル2以降: MCPでツールを追加・システム構築
レベル2から以下のソフトウェアが必要になります：

1. **Docker** 
    - コンテナ化されたツール実行環境
    - [公式サイト](https://www.docker.com/)からダウンロード

2. **Git**
    - バージョン管理システム
    - [公式サイト](https://git-scm.com/)からダウンロード

3. **Visual Studio Code**
    - 統合開発環境（IDE）
    - [公式サイト](https://code.visualstudio.com/)からダウンロード
    - **代替手段**: Sublime Text、Atom、IntelliJ IDEA、メモ帳等でも可

## 🔑 必須アカウント・API Key

### レベル1: 既存エージェントの利用
以下のサービスアカウントが必要です（無料版でも開始可能）：

1. **ChatGPT**
    - OpenAI の対話型AIサービス
    - [ChatGPT](https://chat.openai.com/)でアカウント作成

2. **Manus**
    - Butterfly Effect の対話型AIサービス
    - [Manus](https://manus.im/invitation/HBKSQHDMMP7T)でアカウント作成

### レベル2以降: MCPでツールを追加・システム構築
レベル2から以下のアカウントが必要になります：

3. **Claude**
    - Anthropic の対話型AIサービス
    - [Claude](https://claude.ai/)でアカウント作成

4. **GitHub**
    - コードサンプルのダウンロード・管理
    - [GitHub](https://github.com/)でアカウント作成

### レベル3以降: API Key が必要
レベル3から以下のAPI Keyが必要になります：

1. **OpenAI API Key**
    - [OpenAI Platform](https://platform.openai.com/)で取得
    - カスタムエージェント構築に必要

2. **Claude API Key**
    - [Anthropic Console](https://console.anthropic.com/)で取得
    - カスタムエージェント構築に必要

## 🧬 研究分野別の推奨知識

### 基礎生物学研究者
- **データベース**: PubMed、UniProt、NCBIの基本的な使い方
- **概念**: 遺伝子名、タンパク質名、パスウェイなどの基本用語
- **ファイル形式**: FASTA、GenBank、PDB形式の基本理解

### 創薬研究者
- **データベース**: ChEMBL、DrugBank、PubChemの基本的な使い方
- **概念**: 化合物構造、活性値、薬物動態などの基本用語
- **ファイル形式**: SDF、MOL、SMILES形式の基本理解

### 臨床研究者
- **データベース**: ClinicalTrials.gov、Cochrane Libraryの基本的な使い方
- **概念**: 疾患名、治療法、臨床試験などの基本用語
- **ファイル形式**: CSV、JSON形式の基本理解

## 📚 レベル別の追加要件

### レベル1: 既存エージェントの利用
- **必要なスキル**: 基本的なPC操作のみ
- **使用ツール**: Webブラウザのみ
- **学習時間**: 1-2時間

### レベル2: MCPでツールを追加
- **必要なスキル**: 
    - 設定ファイルの編集
    - Docker の基本的な使用
- **使用ツール**: Docker、Git、テキストエディタ
- **学習時間**: 3-5時間

### レベル3: エージェントシステム構築
- **必要なスキル**: 
    - Python基礎知識
    - コマンドライン操作
    - 簡単なコード理解
    - API Key の管理
- **使用ツール**: 上記 + Python、API Key
- **学習時間**: 1-2週間

### レベル4: マルチエージェントシステム
- **必要なスキル**: 
    - Python プログラミング基礎
    - システム設計の基本理解
    - デバッグスキル
- **使用ツール**: 上記すべて + 高度なPython開発環境
- **学習時間**: 1-2ヶ月
